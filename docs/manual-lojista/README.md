# Manual do Lojista — Lojeo

Bem-vindo ao manual oficial do Lojeo, a plataforma de e-commerce que coloca todo o poder de uma loja virtual moderna nas suas mãos — sem precisar de equipe técnica. Este guia foi escrito pensando em você, lojista, que quer vender mais e gastar menos tempo configurando coisas.

Cada seção traz instruções passo a passo, capturas de tela e dicas práticas. Você não precisa ler tudo em ordem: use o índice abaixo para ir direto ao ponto que precisa.

## Como usar este manual

> 💡 **Dica:** Se você está começando agora, comece pela seção [Primeiros passos](./01-primeiros-passos.md). Em 30 minutos sua loja pode estar pronta para receber o primeiro pedido.

> ⚠️ **Atenção:** Algumas funcionalidades só ficam disponíveis após você conectar integrações externas (gateway de pagamento, transportadora, NF-e). Cada seção indica os pré-requisitos.

> 🚫 **Não faça:** Nunca compartilhe sua senha do painel administrativo, mesmo com colaboradores. Crie usuários separados em [Configurações > Usuários](./06-configuracoes.md).

## Suporte

Se em algum momento você não encontrar resposta neste manual, abra um chamado pelo próprio painel em **Ajuda > Falar com suporte**, ou envie e-mail para `suporte@lojeo.com`. Atendimento de segunda a sexta, das 9h às 18h.
